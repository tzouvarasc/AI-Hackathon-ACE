# Project Overview

## Goal

Build Thalpo as a low-latency voice AI companion that supports elderly users and gives families actionable safety insights.

## Audience

- Primary: elderly users interacting via phone or browser call.
- Secondary: family members/caregivers using the dashboard and alerts.

## Core Message

A calm voice companion can provide daily support while continuously monitoring emotional and cognitive risk signals.

## Constraints

- End-to-end response latency target: under 900 ms.
- Analysis must be non-blocking for the conversation path.
- Greek language support is a required production feature.

## Success Criteria

- Reliable turn-by-turn voice interaction.
- Actionable cognitive/emotion trends in family dashboard.
- Alert quality sufficient to reduce false positives and missed critical events.
